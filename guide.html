מדריך GitHub

הרשמה, יצירת מאגר פרטי, והזמנת שותף — בשלושה שלבים פשוטים

1 הרשמה חינמית ל-GitHub

  1. גלשו ל-github.com
  2. לחצו על Sign up
  3. הזינו כתובת אימייל, בחרו סיסמה ובחרו שם משתמש
  4. עקבו אחר שלבי האימות (פאזל קטן)
  5. פתחו את האימייל שקיבלתם ולחצו על הקישור לאימות
💡 אפשר גם להירשם עם חשבון Google או Apple — פשוט לחצו על הכפתור המתאים בדף ההרשמה.

2 יצירת מאגר (Repository) פרטי

  1. לחצו על כפתור + בפינה הימנית העליונה ← New repository
  2. הזינו שם למאגר (למשל: my-recipes)
  3. סמנו Private (פרטי)
  4. לחצו Create repository

3 הזמנת שותף עם גישה מלאה

  1. בדף המאגר, לחצו על Settings (הגדרות)
  2. בתפריט הצד, לחצו על Collaborators
  3. לחצו Add people
  4. הקלידו את האימייל: haimari1@gmail.com
  5. בחרו את המשתמש מהרשימה ולחצו Add
  6. בחלון שנפתח, בחרו בתפקיד Admin ולחצו על Add to repository
⚠️ חשוב: בחרו בתפקיד Admin (ולא Write או Read) — זה נותן גישה מלאה כולל ניהול הגדרות המאגר, הזמנת שותפים נוספים, ומחיקה.

💡 השותף יקבל אימייל עם הזמנה — צריך ללחוץ על Accept invitation כדי לקבל גישה.
